Welcome to NodCMS installer.

NodCMS is a free multi-lingual CMS based on the popular CodeIgniter PHP framework and can be downloaded on NodCMS.com for free!

This CMS is a good choice for both amateurs and professional web developers, and has made the web creation and developement process so much easier.

The current version, as you see, is the public version and suitable for personal and corporate websites.

You can also visit NodCMS.com for further information.

The installation process is automatic and you will only need to provide the hosting and database information.

Current version is 1.0 and is developed by NodCMS Group.
